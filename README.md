## 🚀 Quick Installation (No Git Required)

1. **Download the latest release**  
   Go to [Releases](https://github.com/KercX/vortex-powerpack/releases) and download `vortex-powerpack.zip`

2. **Extract anywhere** (e.g., `~/vortex-powerpack`)

3. **Set the environment variable**  
   Add this to your `~/.bashrc`, `~/.zshrc`, or system settings:
   ```bash
   export VORTEX_PATH="/absolute/path/to/vortex-powerpack/src"
